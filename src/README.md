# Vela Reference Population Pipeline
This project contains scripts to generate and verify market quality reference data using Claude and Gemini.

## How to run:
1. Setup `.env` with API keys.
2. Run `python pipeline_step1.py`
3. Run `python pipeline_step2.py`
